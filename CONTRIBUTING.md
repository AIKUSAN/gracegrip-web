# Contributing to GraceGrip

Thanks for helping build GraceGrip.

## Principles
- Keep privacy-first behavior
- Keep grace-first tone
- No forced accounts or subscriptions
- Prefer simple, accessible UX

## Setup
1. `npm install`
2. `npm run dev`

## Before Opening a PR
1. `npm run lint`
2. `npm run build`
3. Test core flows on mobile and desktop

## Content Submissions
- Keep language restorative, not shame-based
- Include clear Scripture references
- Keep devotionals concise and practical
